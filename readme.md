# How to create flask app with flask_sqlalchemy

```
    clone this example
    git clone git@github.com:kshyam/flask-blog-with-db.git
```

Install flask_sqlalchemy

```
    pip install flask_sqlalchemy
```

```
flask --debug run
```

OR 

```
 export FLASK_ENV=development
```
